<br>

## 기능키

### 레이어 관련

* `Fn` + `m` = 디폴트 레이어
* `Fn` + `,` = Layer1 (적색 LED)
* `Fn` + `.` = Layer2 (녹색 LED)
* `Fn` + `/` = Layer3 (청색 LED)
* `Fn` + `r`(3초 유지)= 선택한 레이어 초기화 (흰색 LED 두번 깜빡임)
* `Left Alt` + `Right Alt`(3초 유지) = 모든 레이어 초기화 (흰색 LED 다섯번 깜빡임)
<br>

### 매크로 할당

* ___디폴트 레이어에는 매크로를 할당할 수 없다___
* ___매크로 할당 전 딥스위치는 전부 OFF 상태로 두어야 한다___
<br>

1. 원하는 레이어를 선택
1. `Fn` + `Right Ctrl` = 매크로 모드 시작 (흰색 LED)
1. 원하는 단축키 한 번 입력 후 단축키가 수행할 기능 입력
> 예를 들어, `Fn` + `h` 입력 후 "Hello, world!" 를 순서대로 입력
1. `Pn` = 매크로 저장
1. `Fn` + `Right Ctrl` = 매크로 모드 종료 (흰색 LED 소등)
<br>

### 기본 `Pn`, `Fn` 키조합

* `Fn` + `p` = 스크린샷
* `Fn` + `h` = Home
* `Fn` + `n` = End
* `Fn` + `i``k``j``l` = 방향키
* `Fn` + `y` = 계산기
* `Fn` + `z` = 메뉴(마우스 오른쪽버튼)
