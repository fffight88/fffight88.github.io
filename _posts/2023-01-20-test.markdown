# Test Page

## 마크다운 문법
- 큰제목            : ===
- 작은제목          : ---
- ***굵게+이탤릭*** : 별(*) 3개 또는 언더바(__) 3개
- **굵게**          : 별(*) 2개 또는 언더바(__) 2개
- *이탤릭*          : 별(*) 1개 또는 언더바(_) 1개
- ~~취소선~~        : 물결표시(~)  2개
- <u>밑줄</u>       : u태그
- >인용문           : > 기호
- `코드`            : `기호로 감싼다
- 수평 구분선       : *** 또는 --- 또는 ___
- 번호 목록         : 숫자와 점 (1. 등)
- 기호 목록         : - 또는 + 또는 *. Tab키로 계층 변화

## 코드블럭 사용법
```html
<html>
   <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>테스트 페이지</title>
   </head>
   <body>
      <p> Hello, world! 안녕, 세상!</p>      
   </body>
</html>
```

## 링크 사용법
### 참조 링크 : \[텍스트][링크이름] , 다른 줄에 [링크이름]: url
[Myblog][my-blog]

[my-blog]: https://fffight88.github.io
### 인라인 링크 : \[텍스트](url)
[myblog](https://fffight88.github.io)
### url 링크 : 중괄호(<>) 사이에 url
<https://fffight88.github.io>
### 내부 링크 : \[텍스트](#id)
[MyBlog](#my-blog)

## 이미지 사용법 : \![이미지이름]\(url)
이미지는 링크를 삽입하는 방법과 같지만 대괄호 앞에 !를 붙인다.

![이미지이름](url)

## 표 삽입하기

### 구조 잡기

| First Header  | Second Header | Third Header         |
| :------------ | :-----------: | -------------------: |
| First row     | Data          | Very long data entry |
| Second row    | **Cell**      | *Cell*               |
| Third row     | Cell that spans across two columns  ||

```markdown
| First Header  | Second Header | Third Header         |
| :------------ | :-----------: | -------------------: |
| First row     | Data          | Very long data entry |
| Second row    | **Cell**      | *Cell*               |
| Third row     | Cell that spans across two columns  ||
```

### 표 정렬 : 콜론(:)을 헤더의 한쪽 또는 양쪽에 삽입

| Header One | Header Two | Header Three | Header Four |
| ---------- | :--------- | :----------: | ----------: |
| Default    | Left       | Center       | Right       |

```markdown
| Header One | Header Two | Header Three | Header Four |
| ---------- | :--------- | :----------: | ----------: |
| Default    | Left       | Center       | Right       |
```

### 열 병합 : 병합할 컬럼들 사이에 수평구분선 없이 연속으로 파이프기호를 입력

| Column 1 | Column 2 | Column 3 | Column 4 |
| -------- | :------: | -------- | -------- |
| No span  | Span across three columns    |||

```markdown
| Column 1 | Column 2 | Column 3 | Column 4 |
| -------- | :------: | -------- | -------- |
| No span  | Span across three columns    |||
```
