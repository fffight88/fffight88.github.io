#### 당사 ERP 시스템(G7) 유지보수작업 인수 및 회계관련로직 개선, 기능 추가
<ul>
    <li>수행 기간: 2022년 11월 중순 ~ 2023년 1월 초</li>
    <li>수행 배경</li>
        <ul>
            <li>현 시스템의 회계 처리 로직으로 표현하는 데이터의 내용이 현실과 달라 당해연도 내에 해당 부분을 수정하지 않으면 가산세가 부과될 수 있음.</li>
            <li>자회사 합병 및 조직 개편으로 ERP시스템 담당부서가 변경되어 업무를 인수</li>
            <li>기존 담당부서가 다른 프로젝트를 맡아 주력으로 진행해 온 까닭에 사용자의 불편사항이 누적되어 있는 상태임.</li>
        </ul>
    <li>중점 사항: 현금 및 무통장입금 매출과 같은 방식으로 처리되던 외상 매출의 로직을 대폭 변경</li>
    <li>수행 인원: 3명</li>
    <li>본인 프로젝트 기여도: 60%</li>
</ul>

#### 개발 환경
<ul>
    <li>운영체제     : Win11</li>
    <li>플랫폼       : Web</li>
    <li>web서버      : AWS EC2</li>
    <li>로컬서버     : Apache 2.4(XAMPP)</li>
    <li>DB서버       : AWS RDS</li>
    <li>개발언어     : PHP 5.6, JavaScript(Jquery, Ajax)</li>
    <li>프레임워크   : CodeIgnitor 3.0</li>
    <li>텍스트에디터 : Sublime Text</li>
    <li>RDBS        : MySQL</li>
</ul>

#### 프로젝트 수행을 통한 기술적 성장
<ul>
    <li>웹개발: CodeIgnitor 프레임워크를 통해 MVC패턴을 이용한 웹 어플리케이션 구성에 매우 익숙해짐</li>
</ul>

#### 프로젝트 수행의 의미
<ul>
    <li>PHP라는 언어와 IDE가 아닌 텍스트에디터를 이용한 웹개발을 처음으로 해 보는 상황에서 약 1.5개월이라는 길지 않은 기간 내에 회계 로직 제작을 위한 도메인 지식을 함께 공부해서 구현해야 하는 부분이 상당히 도전적이었음.</li>
    <li>교육기관에서 공부를 위해 수행하던 활동 및 프로젝트를 벗어나 처음으로 경험하는 상업적 활동</li>    
</ul>
