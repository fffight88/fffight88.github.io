### 새로운 DB계정 생성

새로운 파트너사 DB연동 등으로 계정을 새로 만들어야 할 때가 있는데, 현재 쿼리 또는 SQLyog의 기능으로 계정을 생성하려 하면 오류가 발생하면서 되지 않는다(테이블 컬럼의 수가 맞지 않다고 하는데 업그레이드를 한참 안해서인 것으로 생각됨). 

1. `tcps_production` DB에 접속
2. `mysql.user` 테이블에 정보를 직접 입력한다.
3. Host = 접속하려는 계정의 외부IP, User = 원하는 계정명, Password = password("원하는 패스워드") 를 입력하고 저장 후 새로고침 해보면 패스워드가 암호화되어 저장되어 있다. 그 후 `FLUSH PRIVILEGES;` 를 실행하여 사용자정보 업데이트
4. 이제 새 연결 메뉴를 열고 MySQL Host Address 에 52.79.207.19, 포트번호에 3306을 입력하고 테스트 연결을 눌러서 'Successful!' 메세지가 뜨면 계정생성은 완료
5. 생성한 계정에 특정 DB에만 접근할 수 있도록 권한을 주고 싶다면 `mysql.db` 테이블에서 Host=접속계정 외부IP, Db=DB명, User=계정명 입력 후 나머지 컬럼들에서 그 DB에 대한 권한을 설정할 수 있는데 주문데이터를 주고받는 정도라면 SIUD 권한 정도만 줘도 괜찮다.

### gnb에 맞춤제품 메뉴 추가하고 제품 등록하기(청년다방, 브레댄코, ...)

> **"좌측에 xxxx 메뉴를 만들고 그 안에 맞춤제품번호 xxxx번(부터 xxxx번)을 추가해주세요"**

1. TCPS 실서버 db의 b2b_icm.tpa001_customize 테이블에서
2. where 조건에 `serial = 제품번호` 를 넣어 select로 검색하면
3. `p_code` 컬럼에서 해당 제품들의 코드 확인
4. `\application\config\config_tcps.php` 에서 `$config['고객사명_tpa013_p_codes']` 배열을 찾는다. 
   *(고객사명: 청년다방=>youngdabang, 브레댄코=>breadnco, ...)*
5. 배열 안에 다음 배열을 추가한다.
```php
array(
	'name' => "원하는 메뉴명",
	'p_code' => array("3번에서 확인한 코드값1", "3번에서 확인한 코드값2", ...)
)
```
<br>
<br>

### gnb의 메뉴명 수정하기

1. TCPS 실서버 db의 b2b_icm.mem_c 테이블에서
2. where 조건에 `c_biz_com = '회사명'` 을 넣어 select로 검색하면
3. `serial` 컬럼에서 해당 고객사의 코드 확인
4. `\application\helpers\function_tcps_helper.php` 에서 `func_tcps_reset_config_code()` 를 찾는다.
5. 해당 고객사의 코드에 해당하는 if문을 찾아 원하는 부분을 수정한다.
6. 청년다방, 브레댄코 등 맞춤제품 부분의 메뉴가 커스터마이징 되어있는 고객사의 맞춤제품(TPA013) 메뉴명은 여기에서 바꿀 수 없고  [좌측 navigation bar에 맞춤제품 메뉴 추가하고 제품 등록하기(청년다방, 브레댄코, ...)](#좌측-navigation-bar에-맞춤제품-메뉴-추가하고-제품-등록하기청년다방-브레댄코-) 를 참조

5. 오류메세지가 몇 줄 뜨지만 무시하고 DB에서 해당 주문을 새로고침해보면 `is_shoot`이 `Y`로 바뀐 것을 확인하고 담당영업매니저에게 확인을 요청한다.

### 명함/맞춤제품에 가변속성값 적용하기(템플릿 생성작업)

1. `제품관리 > 배경샘플 관리` 화면에서 가변으로 적용할 부분을 레이아웃으로 작성할 때 배경이미지(레이어)가 되는 샘플 이미지파일을 업로드
2. `제품관리 > 레이아웃 관리` 화면에서 이 레이어 위에 기존에 생성한 태그, 즉 적절한 타이틀을 가진 입력 텍스트박스를 끌어서 위치를 정하고 각종 속성을 정한 뒤 적용 버튼을 누르면 해당 태그의 속성이 xml파일로 변환된 모습이 화면에 보임. `수정하기` 버튼을 누르면 수정된 내용이 적용된 xml파일이 만들어지고 해당 레이아웃을 사용할 수 있게 된다. 
3. `제품관리 > 템플릿 등록` 화면에서 가변으로 적용할 부분을 제외한, 기본 배경이 되는 (앞면, 뒷면) 이미지파일을 업로드
4. `제품관리 > 템플릿 수정`화면에서 3번에서 등록한 템플릿에서 2번에서 만든 레이아웃을 적용한 후 `템플릿 수정하기`를 누르면 템플릿이 완성된다.
